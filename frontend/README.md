# Regime Trader — Frontend

Next.js 16 (App Router) + TypeScript + Tailwind dashboard for the Regime Trader app.

See the [project README](../README.md) for the full overview, architecture, and run instructions.

```bash
npm install
npm run dev     # http://localhost:3000  (expects the backend API on :8000)
npm test        # Vitest
npm run build   # production build
```
